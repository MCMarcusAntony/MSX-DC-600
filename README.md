# MSX DC-600
Floppy Disk Controller for MSX nad MSX2 Computers.

Talent TDC-600 clone. PCB redrawn for fit to standard cartridge case.

## Compatible Floppy Disk Controller

* WD37C65B (WDC)
* GM82C765B (GoldStar,LGS)
